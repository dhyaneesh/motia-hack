# Knowledge Graph Frontend

Next.js frontend for the Knowledge Graph Chatbot application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file:
```
NEXT_PUBLIC_API_URL=http://localhost:3000
```

3. Run development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3001` (or next available port).

## Features

- Interactive chat interface
- Real-time knowledge graph visualization with React Flow
- Node detail sidebar with references
- Expandable nodes for deeper exploration
